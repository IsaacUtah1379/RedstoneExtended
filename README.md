# Redstone Extended

Extends existing functions of redstone by giving the player more granular control and condensing large circuits into individual blocks. I have multiple blocks I plan to implement, but currently the mod only provides the clock circuit. You can download the mod at Modrinth or CurseForge.

## Clock Circuit
A precisely configurable clock circuit. Allows the player to set how many ticks it should be powered and how many it should be unpowered for each cycle. (Please not that the settings are in **redstone** ticks, not game ticks.)
